UserAppStore

A platform & app store for web apps

Our subscription app store makes it easy to create, discover and use new services and puts all your bills, cancellation and refunding in your control.

On our app store you can publish web applications hosted on your own server or code apps in our project editor.

If you already wrote your web application check out the integration guide for how to become compatible.

Register your free account

By registering you agree to our terms of service, user guidelines and accept our privacy policy.

Code your own single page apps

Our free project editor lets you code single page apps right in your browser. They can be shared, installed, published to the app store, or downloaded to host on your own server!

When you host your single page app yourself it is bundled with Dashboard, with optional modules available for Stripe Subscriptions and Organizations ready to launch as your own SaaS with the same billing and account features.

Panelled interface for using apps

Your installed apps can open side by side, above and below, any combination you want powered by Golden Layout.

On a large, high resolution monitor you might arrange dozens of different web applications comfortably on your screen, especially if care is taken with responsive designs.

App store powered by Stripe Connect

To publish to our app store you must complete a personal or company registration providing relevant contact information that will be verified.

There is no charge for this registration, although there are some Stripe fees if your account is earning money.

We ask for at least 5% of your subscription revenue, if you wish to support our efforts beyond that you may.

Support for organizations

When you subscribe to something in our app store you can do so individually or on behalf of an organization you are in. Organization members can be included in your subscription or setup their own subscription when installing the app themselves.

You can also use organizations to share administrative access to apps published on the app store, shared projects, and imported web servers.

Privacy

There are no tracking, advertisers, social logins or third-party assets collecting your personal information or broadcasting your browsing activity.

However

This website is a portal for using web apps created by others. There are many services and remotely-hosted resources developers may use to improve their apps so third parties may be privvy to some usage information.

Completely open source software

You can deploy our app store software to your own computer or web hosting. The software contains two components, a dashboard server and an application server. They are both NodeJS projects and each supports local file system, Redis, PostgreSQL or S3/Spaces/etc as storage.

The dashboard server provides a complete user account and billing system for web apps, via Dashboard extended with modules for Stripe Subscriptions, Stripe Connect and Organizations.

The application server provides the project editor, app store, app installation and removal process. Dashboard proxies the application server to create a complete site, some content coming from each server. Application servers can be written in any language and use Dashboard's API to access user data.
